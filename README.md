# ARC BP Tracker

Discord login + simple drag and drop tracker for ARC Raiders blueprints.

## Local setup
1) Copy `.env.example` to `.env` and fill values
2) Install deps: `npm i`
3) Init DB: `npx prisma migrate dev --name init`
4) Run: `npm run dev`

## Images
Place blueprint images here:
- public/bps/colored/1.png ... 75.png
- public/bps/gray/1.png ... 75.png
